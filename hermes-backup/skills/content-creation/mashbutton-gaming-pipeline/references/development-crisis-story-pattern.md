# Development Crisis Story Pattern

For Shorts covering games in active development trouble (failed alphas, studio layoffs, reboot/cancel risk, leadership chaos).

## Research Requirements

Before drafting narration, verify from authoritative sources:
- [ ] Named executives installed to oversee project (with titles)
- [ ] Specific studio name and layoff count
- [ ] Financial restructuring context (dollar amounts, reorg names)
- [ ] Root causes cited by insiders (deadlines, planning, leadership rifts)
- [ ] Previous launch timeline now obsolete (explicitly state "dead" or "obsolete")
- [ ] Official silence status (no comment from publisher)

## Source Quality Hierarchy

1. **Insider-Gaming / Tom Henderson** — Industry vet, decent hit rate on Ubi leaks
2. **Game Informer / GI.biz** — Official studio closures confirmed by publisher
3. **Wccftech / aggregator** — Adds memo details to primary reporting
4. **Publisher official** — Zero comment = radio silence (note this)

## Narration Structure for Crisis Stories

**Opening (bridge):** `{Game} {Project Codename} IS IN DEVELOPMENT HELL AND HERE'S WHAT YOU NEED TO KNOW`

**Facts block (plain language, full terms):**
- Alpha failed quality/timeline benchmarks → N executives (names + titles) now oversee daily decisions
- Root causes per insiders: unrealistic deadlines, poor planning, leads ignoring headquarters direction
- Studio {Name} (heritage: {X} years on franchise) ceased game development entirely
- {N} layoffs, now only engine/IT support
- Part of ${X}M cost-cutting and {Reorg Name} reorganization
- Staff fear: reboot (adds years) or cancellation
- Old {Season/Year} launch timeline is dead

**Closing (two-angle, non-binary):**
`WILL {PUBLISHER} COMMIT TO THE {NEW DIRECTION} OR WILL {PROJECT} END UP LIKE {CANCELLED FRAMEWORK} IN THE GRAVEYARD OF CANCELLED {FRANCHISE} GAMES`

## Language Rules (Crisis Stories)

- Spell out: "headquarters" not "HQ", "developers" not "devs", "two hundred million" not "$200M"
- Full names: "Bruno Galet" not "Galet", "Jean-Baptiste Duval" not "Duval"
- "ceased game development entirely" not "shifted to support"
- "ignored headquarters and did their own thing" not "ignored HQ and went rogue"
- "commit to the military simulation direction" not "double down on mil sim"
- Word count target: 90-100 words (crisis stories need more facts)

## Fact-Checking Checklist

- [ ] Alpha failure confirmed (not just "delayed")
- [ ] Executive names and titles verified
- [ ] Studio name, location, heritage years correct
- [ ] Layoff count from GI.biz or equivalent
- [ ] Dollar amount and reorg name from official announcement
- [ ] Previous timeline explicitly marked obsolete
- [ ] No unconfirmed rumors presented as fact
